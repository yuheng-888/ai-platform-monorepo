# Resin v1.0.0 Migration Guide (Auth Version Switch)

This guide is for users upgrading from older versions to v1.0.0. The goal is to migrate auth formats smoothly and avoid startup failures or authentication errors.

## 1. Make Sure the Service Can Start First (Compatibility Mode)

Starting from v1.0.0, the `RESIN_AUTH_VERSION` environment variable is required.

If you are upgrading directly from an older version, start with compatibility mode first:

```bash
RESIN_AUTH_VERSION=LEGACY_V0
```

This keeps legacy auth behavior unchanged so the service can recover first, then you can continue migration safely.

## 2. Clean Up Incompatible Configurations

### 2.1 Check and Fix `RESIN_PROXY_TOKEN`

Before switching to `V1`, `RESIN_PROXY_TOKEN` must not contain:

- `.:|/\@?#%~`
- spaces, tabs, line feeds, or carriage returns

### 2.2 Check and Fix Platform Names

Before switching to `V1`, make sure your Platform names do not:

- contain `.:|/\@?#%~`
- contain spaces, tabs, line feeds, or carriage returns
- match reserved keywords

If historical Platform names are invalid, rename them under `LEGACY_V0` first, then switch to `V1`.

## 3. Switch to V1

After cleanup, change the environment variable to:

```bash
RESIN_AUTH_VERSION=V1
```

After restart, if any incompatible item remains, Resin will refuse to start and report an error.

## 4. Client Auth Format Changes

### Forward Proxy

- Old format (`LEGACY_V0`): `RESIN_PROXY_TOKEN:Platform:Account`
- New format (`V1`): `Platform.Account:RESIN_PROXY_TOKEN`

`curl` example (`V1`):

```bash
curl -x http://127.0.0.1:2260 \
  -U "Default.user_tom:my-token" \
  https://api.ipify.org
```

### Reverse Proxy

- Old format (`LEGACY_V0`): `/token/Platform:Account/protocol/host/path`
- New format (`V1`): `/token/Platform.Account/protocol/host/path`, or provide Account using the `X-Resin-Account` request header.

`curl` example (`V1`):

```bash
# Provide Account in URL
curl "http://127.0.0.1:2260/my-token/Default.user_tom/https/api.ipify.org"

# Provide Account via X-Resin-Account header
curl "http://127.0.0.1:2260/my-token/Default/https/api.ipify.org" -H "X-Resin-Account: user_tom"
```
